Emacsの暗背景化

Emacs のカラー表示が見にくいなと思ったら、 .emacs もしくは .emacs.el ファイルの中から 

 (cond (window-system
        (setq hilit-mode-enable-list  '(not text-mode)
              hilit-background-mode   'light
              hilit-inhibit-hooks     nil
              hilit-inhibit-rebinding nil)
 
        (require 'hilit19)
        ))

こんなこんなかんじのところを探します。 あくまで、このようなかんじのところです。 そしたら、以下のようにします。 

 (cond (window-system
        (setq hilit-mode-enable-list  '(not text-mode)
              ;hilit-background-mode   'light
              hilit-background-mode   'dark
              hilit-inhibit-hooks     nil
              hilit-inhibit-rebinding nil)
 
        (require 'hilit19)
        ))

「;」 (セミコロン) から行末まではコメントです。
