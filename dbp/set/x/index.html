X Window System の暗背景化

Xは暗背景党に配慮されています。 

xterm -rv

このように -rv オプションを付けるだけで、 とりあえずは暗背景化されます。

しかし、毎度 -rv オプションを付けるのは大変なので、 ホームディレクトリ内の .Xdefaults (または、 .Xresources) ファイル に以下の1行を追加します。 

*reverseVideo:on

こうすれば、今後起動する X クライアントは暗背景化されます。 場合によっては、 

xrdb .Xdefaults

と、唱える必要があります。 なにはともあれこれで、 editres, xfontsel, xcalc, xterm などは暗背景化されます。
